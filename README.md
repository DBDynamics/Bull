# DBD-Bull

## About Bull
- Bull是DBD团队开发的一款行星减速直流无刷伺服电机， 超级紧凑，拥有双供电接口，双通信接口，双限位接口，泄流电阻接口以及刹车接口。

## 特性
- 体积小
- 扭力大
- 接口丰富
- 电压稳定
- 刹车接口
## 参数
工作电压 24V
额定扭矩 5Nm
峰值扭矩 11Nm
最大输出转速 120rmp
